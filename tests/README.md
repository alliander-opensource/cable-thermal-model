# (Py)Test suite
This directory tests organized into subdirectories. Shared objects are defined in `conftest.py`.

## Running tests

Run tests from the terminal using `pytest tests/` while your conda environment is active.

## Test organization

Tests are organized similar to how the repository itself is structured. E.g. tests for the Model are found in
test_model.py in tests/model.
